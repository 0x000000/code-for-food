== Code-for-Food

Приложение для заказа обедов в офис. Используется в компании {Altoros Inc.}[http://altoros.com] с
30 матра 2011. Если у тебя есть свободное время, желание поучавствовать
в разработке и идея - Welcome! Fork => Add New Feature => Pull Request :)

Список тасков в пивотале!
